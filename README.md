# Voice Assistant using Python

A simple Python-based voice assistant that takes user commands (typed input),
performs basic tasks, and responds using text-to-speech.

This project is built for learning purposes and demonstrates how a basic
assistant works using Python libraries.



## Features
- Greets the user based on time (Morning / Afternoon / Evening)
- Searches and reads summaries from Wikipedia
- Opens Google and YouTube in the browser
- Tells the current time and date
- Tells random jokes
- Responds using text-to-speech
- Handles unknown or empty commands gracefully



## Technologies Used
- Python
- pyttsx3 (Text-to-Speech)
- Wikipedia API
- pyjokes
- datetime
- webbrowser
